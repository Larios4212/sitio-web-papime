Acerca del Proyecto PAPIME

Este proyecto tiene como finalidad desarrollar recursos educativos innovadores que faciliten el aprendizaje de conceptos complejos en geología, mineralogía y geomorfología mediante el uso de tecnologías inmersivas y modelos tridimensionales.

Objetivo

Crear un entorno educativo virtual que permita a estudiantes y profesores explorar conceptos geológicos de manera interactiva, mejorando la comprensión espacial y facilitando el aprendizaje activo mediante la visualización de modelos 3D y experiencias inmersivas.
